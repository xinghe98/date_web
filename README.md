# date_web
数据自动可视化web项目

# 目录介绍
- conf
    - 存储配置文件 
- controller
    - 路由控制器
- dao
    - 数据库初始化与连接 
- models
    - 模型定义 
- response
    - 响应封装 
- router
    - 路由配置 
- service
    - 控制器的主要逻辑 
- util
    - 其他工具 
# 说明
- 所有时间格式使用`YYYY-MM-DD`的格式
